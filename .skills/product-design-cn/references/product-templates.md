# 产品设计产出模板参考

## 1. 创意发散模板

```markdown
# 设计方案：{项目名称}

## 需求概述
- 产品：{产品名称}
- 目标用户：{用户画像}
- 核心功能：{功能描述}

## 方向一：{方案名称}
- **设计理念**：{核心理念}
- **布局思路**：{布局方案描述}
- **适用场景**：{适用场景说明}
- **关键页面**：{页面清单}

## 方向二：{方案名称}
- **设计理念**：{核心理念}
- **布局思路**：{布局方案描述}
- **适用场景**：{适用场景说明}
- **关键页面**：{页面清单}

## 方向三：{方案名称}
- **设计理念**：{核心理念}
- **布局思路**：{布局方案描述}
- **适用场景**：{适用场景说明}
- **关键页面**：{页面清单}

## 推荐方案
{推荐的方向及理由}
```

## 2. UX 审查报告模板

```markdown
# UX 审查报告：{产品/页面名称}

## 审查概况
- 审查范围：{范围}
- 审查日期：{日期}
- 严重程度定义：P0(阻塞) / P1(严重) / P2(一般) / P3(建议)

## 问题清单

| 严重程度 | 问题描述 | 位置 | 建议修复 |
|:---:|:---|:---|:---|
| P0 | {问题} | {位置} | {建议} |
| P1 | {问题} | {位置} | {建议} |
| P2 | {问题} | {位置} | {建议} |
| P3 | {问题} | {位置} | {建议} |

## 各维度评估

| 维度 | 评分 | 说明 |
|:---|:---:|:---|
| 信息架构 | {评分}/5 | {说明} |
| 导航流程 | {评分}/5 | {说明} |
| 交互反馈 | {评分}/5 | {说明} |
| 内容清晰度 | {评分}/5 | {说明} |
| 一致性 | {评分}/5 | {说明} |

## 总结
{总体评价和关键建议}
```

```markdown
| 键盘导航 | {描述} | 2.1.1 | 高/中/低 | {建议} |
| 屏幕阅读器 | {描述} | 4.1.2 | 高/中/低 | {建议} |
| ARIA 标签 | {描述} | 4.1.2 | 高/中/低 | {建议} |

## 总结
{总体可访问性评估和优先修复建议}
```
