## 4. Pitch Deck 大纲模板

```markdown
# Pitch Deck：{项目名称}

1. **封面** — 交易名称、日期、团队
2. **执行摘要** — 交易亮点和核心推荐
3. **市场概览** — 行业趋势、市场规模、增长驱动
4. **交易能力** — 团队经验、过往案例、行业专长
5. **交易案例** — 2-3 个代表性交易
6. **目标公司分析** — 公司概览、财务状况、估值
7. **交易结构建议** — 对价形式、融资方案、时间表
8. **团队介绍** — 核心成员背景
9. **附录** — 详细财务数据、免责声明
```

## 5. CIM 结构模板

```markdown
# 保密信息备忘录：{公司名称}

## 章节结构
1. **执行摘要**
2. **公司概览** — 历史沿革、业务模式、竞争优势
3. **行业分析** — 市场规模、增长驱动、竞争格局
4. **产品/服务** — 产品线、技术优势、客户结构
5. **销售与营销** — 渠道、定价、客户获取
6. **财务分析** — 历史财务、关键指标、质量分析
7. **预测** — 收入预测、成本结构、EBITDA
8. **管理层** — 核心团队背景
9. **数据室索引**
10. **附录**
```
