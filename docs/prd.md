# 需求文档

## 1. 应用概述

**应用名称**：自动跟读听书（书卷）

**应用描述**：一款支持大文档导入、多音色朗读、自动跟读高亮、智能背景音乐匹配的沉浸式听书移动端应用，采用中国风/国学风格设计，为用户提供温润书卷气息的听书体验。

## 2. 用户与使用场景

**目标用户**：喜欢听书、希望通过跟读提升阅读体验的移动端用户

**核心使用场景**：
- 用户导入本地TXT、PDF等格式的长篇文档
- 系统自动根据文章内容匹配音色、背景音乐、音效，打造沉浸式听书体验
- 用户在阅读器中跟随高亮文字进行跟读练习
- 用户通过播放控制调整朗读进度、语速

## 3. 页面结构与功能说明

### 页面层级结构

```
应用首页（底部Tab导航）
├── 书库
│   ├── 书籍列表
│   └── 书籍详情页
│       └── 阅读器页面
└── 设置
```

### 3.1 书库页面

**功能说明**：
- 展示已导入的书籍列表，包含书名、封面（如有）、导入时间
- 支持导入本地文档文件（TXT、PDF格式）
- 点击书籍进入书籍详情页

### 3.2 书籍详情页

**功能说明**：
- 展示书籍基本信息（书名、作者、简介等）
- 展示章节列表，用户可选择章节进入阅读器
- 显示当前朗读进度（如已读至第X章）

### 3.3 阅读器页面

**功能说明**：
- **文字展示区域**：
  - 展示当前章节文字内容
  - 支持字体大小调节
  - 朗读时当前朗读句子高亮显示
  - 文字自动滚动跟随朗读进度

- **播放控制栏**：
  - 播放/暂停按钮
  - 上一句/下一句按钮
  - 进度条，显示当前朗读进度，支持拖动调整
  - 语速调节（如0.5x、1.0x、1.5x、2.0x）

- **沉浸式体验**：
  - 音色自动匹配：系统根据文章内容/场景自动选择合适音色
  - 背景音乐自动切换：根据内容自动切换背景音乐
  - 音效自动触发：根据情节自动触发音效（如风声、雨声、鸟鸣、钟声等）

### 3.4 设置页面

**功能说明**：
- 字体大小默认值：用户可设置阅读器默认字体大小
- 主题设置：支持亮色/暗色主题切换
- 语速默认值：用户可设置默认朗读语速
- 沉浸模式开关：开启后全自动控制音色/背景音乐/音效，关闭后可手动调整

## 4. 业务规则与逻辑

### 4.1 文档导入与处理
- 支持导入TXT、PDF格式的本地文档
- 大文档分段处理，每段约200-500字
- 导入后自动解析章节结构（以「第X章/回/节」等标题为分隔）

### 4.2 朗读与跟读
- 使用MiniMax TTS接口（通过Edge Function调用）进行语音合成
- 朗读时当前句子高亮显示，文字自动滚动
- 用户可通过播放控制栏调整朗读进度、语速

### 4.3 智能沉浸式体验
- 接入GLM-4-Flash模型（https://open.bigmodel.cn/api/paas/v4，API Key: 9f9a917e3cbf480291bfe80b2d8ed744.8RW6t1WfbnNJz7Oo）进行内容分析
- 模型智能分析文档内容，自动决定：
  - 当前段落使用什么音色
  - 当前场景搭配什么背景音乐
  - 何时触发什么音效
- 背景音乐和音效使用预置的音频资源或网络音频

### 4.4 朗读进度持久化
- 朗读进度自动保存至本地数据库
- 用户下次打开应用时可继续上次朗读位置
- 进度包含书籍、章节、句子位置

### 4.5 数据存储
- 使用expo-sqlite存储书籍、章节、朗读进度等数据
- 所有数据存储在本地，无需用户登录

### 4.6 UI风格
- 中国风/国学风格，温润书卷气息
- 暖色调（米色、棕色、金色系）
- 优雅的书法感字体搭配
- 底部Tab导航：书库、设置

## 5. 异常与边界情况

| 场景 | 处理方式 |
|------|----------|
| 导入文档格式不支持 | 提示用户仅支持TXT、PDF格式 |
| 导入文档解析失败 | 提示用户文档格式异常，无法导入 |
| 朗读过程中网络异常 | 暂停朗读，提示用户检查网络连接 |
| 背景音乐加载失败 | 关闭背景音乐，提示用户背景音乐加载失败 |
| 模型API调用失败 | 使用默认音色/背景音乐继续朗读，提示用户智能匹配功能暂时不可用 |
| 用户拖动进度条时正在朗读 | 暂停朗读，跳转至拖动位置后继续朗读 |
| 沉浸模式关闭时 | 允许用户手动调整音色/背景音乐/音效 |
| PDF文档无法提取文本 | 提示用户该PDF文档无法解析，建议使用TXT格式 |

## 6. 验收标准

1. 用户打开应用，进入书库页面
2. 用户点击导入按钮，选择本地TXT或PDF文档，导入成功后书籍出现在书库列表中
3. 用户点击书籍进入书籍详情页，选择章节进入阅读器页面
4. 用户点击播放按钮，应用开始朗读，当前句子高亮显示，文字自动滚动，系统自动匹配音色、背景音乐、音效
5. 用户点击暂停按钮，朗读暂停，再次点击播放按钮继续朗读
6. 用户调整语速，朗读速度变化
7. 用户关闭应用，再次打开应用进入书库页面，点击书籍进入阅读器页面，朗读进度保持在上次位置
